# Lancolt Lista

Készítsen egy láncolt listát ami mindkét irányba bejárható, valósítsa meg a következő metódusokat:
 * Elejére Beszúrás
 * Végére Beszúrás
 * Törlés
 * A lista legyen mindkét irányba bejárható
    * IEnumerable oda
    * Bejáró vissza
 * Rendezve beszúrás
 ```c#
 class DoubleLinkedList<T> where T: IComparable 
 ```
